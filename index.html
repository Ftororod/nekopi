🧠 AI-Assisted Analysis

NekoPi anonymizes all network data locally before any analysis — IPs, hostnames, and identifiers are replaced with generic labels. The anonymized data is then sent to the configured AI engine for diagnosis.

☁️ Gemini (Cloud)
Google Gemini via API. Requires an API key configured in Settings. Data is anonymized before leaving the device. Best for complex analysis and natural language explanations.
🖥️ Ollama (Local)
100% on-device inference via Ollama. Supports llama3, mistral, phi3 and others. No data leaves the RPi5. Best for sensitive environments where cloud access is not permitted.
